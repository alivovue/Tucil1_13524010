## Deskripsi Singkat
Program ini adalah **Queens Solver** berbasis **JavaFX** untuk menyelesaikan puzzle ratu pada papan **NÃ—N** dengan **papan berwarna**. Algoritma utama yang digunakan pada program ini adalah **Exhaustive Search**.
Pada program ini, user dapat :
- load board dari file `.txt`,
- edit board langsung lewat mode edit,
- menjalankan brute force sambil melihat progress,
- download gambar hasil (PNG).

---

## Cara Menjalankan

### Prasyarat
- Java **17+** (disarankan)
- Maven terinstall

### Download Maven
Untuk mendownload maven, pengguna bisa mengikuti arahan dari website resmi Apache Maven berikut : 
https://maven.apache.org/install.html

### Jalankan dengan Maven
Masuk ke dalam folder my-app, lalu compile dan jalankan program dengan command berikut : 
```bash
cd my-app
mvn package
mvn javafx:run
```
